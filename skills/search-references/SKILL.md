## 概要
提示された名前の変更案（`suggested_name`）について、対象の呼び出し元や参照箇所をひとつ残らず特定するため、`grep` や `findstr` 等の CLI ツールを駆使してワークスペース全体を検索するスキルです。

## 使用手順
ターミナルを使用して `grep` または `findstr` で検索を実行してください。

### 検索対象の厳守事項
1. **全体検索:** 単なる `.cpp` や `.h` だけでなく、Visual Studio ソリューションファイル (`.sln`) やプロジェクトファイル (`.vcxproj`)、Makefile、CMakeLists.txt、その他リソースファイルも含めて、システム全体で名前が使用されている箇所を漏れなく検索してください。
2. **単語境界の厳守:** 対象の識別子を検索する際、必ず単語境界（Word Boundary）として一致するものだけを対象としてください。
3. **文字化けコメントの回避:** 文字化けテキスト内に偶然ターゲット文字列が含まれていても、絶対に置換対象（`action`）に含めないでください。
4. **エンコーディングのパススルー:** 呼び出し元から提供された `fileencoding` を、必ず自身の出力するJSONにもそのまま引き継いでください。

### 出力フォーマット（厳守）
純粋なJSON配列のみを出力してください。
[
  {
    "file": "path/to/source.h",
    "fileencoding": "Shift-JIS",
    "line": 45,
    "action": "replace calcData with CalculateData"
  }
]
