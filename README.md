# dot-github-files

このプロジェクトは、GitHub Copilot 向けのサンプルの Prompt（プロンプト）、Agent（エージェント）、および Skill（スキル）を定義するものです。AI 開発のベストプラクティスに基づき、効果的な実装例を提供することを目的としています。

## フォルダ構成について

このリポジトリの主要なフォルダ構成は以下の通りです。

*   **`prompts/`**: 特定のタスクや指示を与えるためのプロンプト定義ファイルを格納しています。
*   **`skills/`**: エージェントが実行する具体的なツールや関数の定義ファイルを格納しています。
*   **`spec/`**: プロンプト、エージェント、スキルの作成ルールとベストプラクティスを定めた仕様書（例：`agent-spec.md`）を格納しています。

各要素の詳細な作り分け方や命名規則については、ルートディレクトリの [`Agent.md`](./Agent.md) を参照してください。
